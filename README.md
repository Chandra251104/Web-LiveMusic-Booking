# Diamon Music Booking System

Website booking live music untuk acara cafe, wedding, gathering, dan private event.

## Fitur
- Pemilihan paket Duo, Trio, dan Full Band
- Pemilihan penyanyi dan musisi
- Booking jadwal acara
- Perhitungan biaya otomatis
- Estimasi DP 50%
- Responsive design

## Teknologi
- React JS
- Vite
- Tailwind CSS
- JavaScript

## Cara Kerja
Pengguna memilih paket musik, memilih musisi yang tersedia, menentukan tanggal acara, dan melihat estimasi biaya secara otomatis.
